# 🧬 GeneJelly – DNA Data Storage Platform

GeneJelly is a futuristic frontend web application that simulates storing digital data inside DNA sequences.

It converts binary data into DNA base pairs (A, T, G, C) using a 2-bit mapping system and provides encoding and decoding functionality through a modern UI.

---

## 🚀 Features

- 🔐 DNA Encoding (Binary → A/T/G/C)
- 🔓 DNA Decoding (A/T/G/C → Binary)
- 📊 Analytics Section
- 📄 Documentation Section
- 🌙 Dark Themed UI
- 🧬 Custom DNA Branding
- 💻 Fully Frontend (No Backend Required)

---

## 🧠 How It Works

DNA base pair mapping:

| Binary | DNA |
|--------|-----|
| 00     | A   |
| 01     | T   |
| 10     | G   |
| 11     | C   |

The application:
1. Converts file data into binary
2. Maps binary pairs into DNA bases
3. Displays encoded DNA sequence
4. Supports reverse decoding
